当用户请求运行 step_1.py 脚本时，你需要：
1. 从用户消息中提取脚本路径（可能包含 .py 后缀）
2. 识别以下可选参数（如有提及）：
   - 输入文件1（如“第一个输入文件”、“牙根工作表”）：对应 --input1
   - 输入文件2（如“参考名录”）：对应 --input2
   - 输出文件（如“保存到”、“输出路径”）：对应 --output
   - 国家保护级别（如“国家一级、二级”）：对应 --national_levels，多个用逗号分隔
   - 省级保护（如“省级：四川”）：对应 --regional_level
   - 居留型标记（如“pa=ⅤA”）：对应 --pa
3. 调用 run_python 工具，其中 script_path 为脚本路径，keywords 为 ["--参数名", "参数值", ...] 的列表。
如果用户未明确关键词，你可以向用户询问或传递空数组。
